# Instagram Clone - React 앱

React로 만든 Instagram 스타일의 피드 앱입니다.

## 설치 및 실행

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:5173` 접속

## 사용된 이미지

- 피드 이미지: `Images/img01.png` ~ `Images/img10.jpg`
- 프로필 사진: `Images/man.png`, `Images/woman.png`

## 기능

- Instagram 스타일의 피드 레이아웃
- 좋아요 기능 (클릭 시 하트 아이콘 변경)
- 반응형 디자인 (모바일 지원)
- 10개의 샘플 피드 게시물
